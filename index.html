TECNOLOGIAS

HTML

HTML é a linguagem de marcação padrão para criar páginas da web. É a espinha dorsal de qualquer site que você acessa na internet. É a linguagem que dá forma e estrutura ao conteúdo que você vê em qualquer página web.

CSS

É uma linguagem de estilo que você usa para controlar a aparência e o layout de uma página web escrita em HTML. Enquanto o HTML estrutura o conteúdo, o CSS define como esse conteúdo é exibido.

JAVASCRIPT

JavaScript é uma linguagem de script que adiciona funcionalidades às páginas da web. Ele permite, por exemplo, manipular janelas do navegador.

GIT

O Git é um sistema de controle de versão distribuído amplamente utilizado no desenvolvimento de software. Ele permite que desenvolvedores gerenciem mudanças no código-fonte ao longo do tempo, facilitando a colaboração em equipe e o rastreamento de alterações.

GITHUB

O GitHub é uma plataforma de hospedagem de código-fonte que utiliza o sistema de controle de versão Git. Ele é amplamente utilizado por desenvolvedores para colaborar em projetos, tanto privados quanto de código aberto.

BOOTSTRAP

O Bootstrap é um framework front-end amplamente utilizado para o desenvolvimento de sites e aplicações web responsivas. Ele facilita a criação de layouts bonitos e funcionais de forma rápida e eficiente
